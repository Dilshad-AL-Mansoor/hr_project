## Description
This project aims to analyze employee data using SQL and visualize insights using Power BI. The goal is to identify trends, salary averages, and employee performance metrics for HR decision-making.

## Tools Used
- MySQL (for SQL queries)
- Power BI (for data visualization)
- Excel (for quick analysis)

## Getting Started
1. Install MySQL and Power BI.
2. Import `employee_data.csv` into MySQL.
3. Run SQL queries from `SQL_Scripts/` to analyze data.
4. Load the filtered data into Power BI from `Exported_Data/`.
5. Explore the dashboard and insights.

## Key Insights
- Department-wise average salary
- Employee attrition analysis
- Performance vs. promotion trend
